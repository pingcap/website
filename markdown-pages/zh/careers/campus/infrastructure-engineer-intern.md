# 数据库开发实习生

**职位描述：**

你能从工作中学习到什么？
 
- 如何构建一个分布式关系数据库；
 
- 如何将其包装成为一套完整的商业产品；
 
- 亲身参与以上过程，并实践你所掌握的开发技术； 

**任职要求：**

1. 熟悉常用的开发语言，熟悉 Golang/Rust 优先；

2. 熟悉分布式系统/数据库系统优先；

3. 有开源项目实践经历优先；

4. 实习优秀者有机会转正，并有机会获得期权。
